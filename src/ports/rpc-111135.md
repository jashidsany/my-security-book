# RPC (111/135)

### Enum RPC services

```bash
rpcinfo -p $IP
```

> List registered RPC services

**Why:** Find NFS, mountd, etc

<details>
<summary>Example Output</summary>

```
rpcinfo -p 10.10.10.5
program vers proto  port
100000    4   tcp    111  portmapper
100003    3   tcp   2049  nfs
100005    3   tcp  34567  mountd
(NFS available! Check showmount)
```
</details>

---

### rpcclient null session

```bash
rpcclient -U '' -N $IP
```

> Anonymous RPC connection

**Why:** Enum users, groups, shares

<details>
<summary>Example Output</summary>

```
rpcclient -U '' -N 10.10.10.5
rpcclient $> enumdomusers
user:[Administrator] rid:[0x1f4]
user:[Guest] rid:[0x1f5]
user:[svc_backup] rid:[0x451]
```
</details>

### Enum domain users (rpcclient)

```bash
rpcclient $> enumdomusers
```

> After connecting

**Why:** Full user list

<details>
<summary>Example Output</summary>

user:[Administrator] rid:[0x1f4]
user:[john.smith] rid:[0x451]

### Enum domain groups

```bash
rpcclient $> enumdomgroups
```

> Group membership

**Why:** Find admin groups

<details>
<summary>Example Output</summary>

```
rpcclient $> enumdomgroups
group:[Domain Admins] rid:[0x200]
group:[Domain Users] rid:[0x201]
group:[IT-Support] rid:[0x450]

rpcclient $> querygroupmem 0x200
rid:[0x1f4] (Administrator)
rid:[0x451] (john.smith is DA!)
```
</details>

### User info

```bash
rpcclient $> queryuser <rid>
```

> Detailed user info

**Why:** Account details

<details>
<summary>Example Output</summary>

```
rpcclient $> queryuser 0x451
  User Name: john.smith
  Full Name: John Smith
  Description: IT Admin
  Logon Time: Mon, 24 Feb 2026
```
</details>
